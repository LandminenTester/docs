# Brew Society: Comprehensive Society Management

Brew Society provides a robust system for creating and managing societies. It integrates seamlessly with syn_shops, allowing for a comprehensive society experience.

## Features

- **Create and manage societies** with ease
- **Compatible with syn_shops** for extended functionality
- **Server-side only**
