# Brew Cases: Unlimited Case Design and Opening

Brew Cases offers a dynamic and engaging case-opening experience inspired by popular games like CSGO. With the ability to design unlimited cases, this system provides endless possibilities for customization and player engagement.

## Features

- **Unlimited case design** for endless possibilities
- **Random case-opening** experience
- **Inspired by CSGO** for familiar gameplay
- **Server-side only**

You can find the source code and contribute to the project on [GitHub](https://github.com/YourRepo/brew_cases).
